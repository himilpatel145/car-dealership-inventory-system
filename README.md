# Car Dealership Inventory System 🚗

## Project Overview

Car Dealership Inventory System is a full-stack web application developed to manage vehicle inventory for a car dealership.

The application allows users to register, login, browse available cars, search vehicles, view car details, and submit enquiries.

Admin users can manage dealership inventory by adding, updating, deleting vehicles, and managing enquiries.

This project is built using the MERN stack with JWT authentication and a responsive React frontend.

---

# Features

## User Features

- User Registration
- User Login
- JWT Token Authentication
- Browse available vehicles
- Search and filter cars
- View detailed car information
- Submit enquiry to dealership


## Admin Features

- Admin authentication
- Add new vehicles
- Upload vehicle images
- Update vehicle details
- Delete vehicles
- View customer enquiries
- Manage inventory


## API Endpoint Note

The project implements vehicle inventory functionality using the /api/cars endpoint.
This provides CRUD operations, authentication, and enquiry management.
